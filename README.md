# \# Cloud Expense Tracker API

# 

# Cloud-native expense tracker API built with \*\*Quarkus\*\* and \*\*Java 17\*\*.  

# This project is designed as a portfolio application to demonstrate backend, cloud and DevOps skills (GCP + AWS).

# 

# \## Tech Stack

# 

# \- Java 17

# \- Quarkus (RESTEasy Reactive, JSON, OpenAPI, Health)

# \- REST API (CRUD) for managing expenses

# 

# \## Endpoints (v1 - in-memory)

# 

# \- `GET /api/expenses` – List all expenses

# \- `GET /api/expenses/{id}` – Get a single expense by id

# \- `POST /api/expenses` – Create a new expense

# \- `PUT /api/expenses/{id}` – Update an existing expense

# \- `DELETE /api/expenses/{id}` – Delete an expense
